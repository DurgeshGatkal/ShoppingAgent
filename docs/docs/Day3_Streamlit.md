Today's task is create a UI using Streamlit.

What is Streamlit?
Streamlit is a Python framework used to create web applications quickly without writing HTML, CSS, or JavaScript.

Instead of designing webpages manually, we write Python code and Streamlit automatically creates the interface.


Why are we using Streamlit?
For BuySense AI, we need a frontend where users can:

Type shopping queries.
View AI responses.
Compare products.
Upload documents (later).
View product cards (later).

Instead of learning frontend development now, Streamlit lets us focus on AI.
